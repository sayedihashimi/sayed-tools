Usage: dotnet new --uninstall template-spec [options]

Template spec:        Identifies the template to uninstall.
                      For a nuget package, specify the packag name.
                      For a folder use, specify the folder path.
Options:
  -h, --help          Show help information.

Examples:
    dotnet new --uninstall Microsoft.AspNetCore.SpaTemplates
    dotnet new --uninstall c:\projcet-templates
