.NET Template Generator

ASP.NET Core Web API Template
Author: Microsoft

Usage: dotnet new webapi [options]

Options:                                                                             
  -n|--name                       The name for the output being created. If no name is specified, the name of the current directory is used.

  -o|--output                     Location to place the generated output.

  -lang|--language                Specifies the language of the template to create.

  -au|--auth            The type of authentication to use                                                                       
                            None         - No authentication                                                                    
                            SingleOrg    - Organizational authentication for a single tenant                                    
                            Windows      - Windows authentication                                                               
                        Default: None                                                                                           

  -s|--aad-instance     The Azure Active Directory instance to connect to (use with Single-org auth).                           
                        string - Optional                                                                                       
                        Default: https://login.microsoftonline.com/                                                             

  -A|--Audience         The AppIdUrl for the project (use with Single-org auth).                                                
                        string - Optional                                                                                       
                        Default: https:////qualified.domain.name/Company.WebApplication1                                        

  --client-id           The Client ID for this project (use with Single-org auth).                                              
                        string - Optional                                                                                       
                        Default: 11111111-1111-1111-11111111111111111                                                           

  --domain              The domain owning the directory (use with Single-org auth).                                             
                        string - Optional                                                                                       
                        Default: qualified.domain.name                                                                          

  --tenant-id           The TenantId ID of the directory to connect to (use with Single-org auth).                              
                        string - Optional                                                                                       
                        Default: 22222222-2222-2222-2222-222222222222                                                           

  -r|--org-read-access  Whether or not to allow this application read access to the directory (only applies to Single-org auth).
                        bool - Optional                                                                                         
                        Default: false                                                                                          

  -f|--framework        The target framework for the project.                                                                   
                            netcoreapp1.0                                                         
                            netcoreapp1.1                                                         
                            netcoreapp2.0                                                         
                        Default: netcoreapp2.0                                                                                  

  --no-restore          If specified, skips the automatic restore of the project on create.                                     
                        bool - Optional                                                                                         
                        Default: false
