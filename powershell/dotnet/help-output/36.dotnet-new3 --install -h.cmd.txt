Usage: dotnet new --install template-spec [options]

Template spec:        Identifies the template to install.
                      For a nuget package,specify "packageName::version", you can use * for version to indicate latest.
                      For a folder use, specify the folder path.
Options:
  -h, --help          Show help information.

Examples:
    dotnet new --install -h
    dotnet new --install "Microsoft.AspNetCore.SpaTemplates::*"
    dotnet new --install c:\projcet-templates
